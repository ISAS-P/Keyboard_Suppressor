◇ 概要 ◇
	ボイスチャットなどで、キーボードの打鍵音が通話に入ることを防ぐプログラムです。
	キーの打鍵を監視し、打鍵があれば音声をミュートします。
	打鍵終了後、任意の待ち時間だけ待機してミュートを解除します。
	打鍵監視により音声をミュートする都合上、打鍵中に音声は流れません。

◇ 動作条件 ◇
	Windows10 Home(64bit)での動作を確認しています。
	恐らく他のWindows（8,7など）でも動作するとは思いますが、自己責任でお願いします。

◇ つかいかた ◇
	プログラムを起動すると、ターミナルが立ち上がります。
	画面の指示に従って入力デバイス、出力デバイス、キー打鍵後の待ち時間[秒]を指定してください。
	待ち時間は0.5[秒]ほどあれば十分だと思いますが、状況に合わせて指定してください。
	終了時はウィンドウの閉じるボタンか、Ctrl+Cで終了してください。

	このプログラムとボイスチャットとの接続に関しては、
	出力デバイスに仮想サウンドカードを指定するなどしてください。
	マイク（入力デバイス） => Keyboard_Supressor => 仮想サウンドカード（出力デバイス） => ボイスチャット
	のように接続してください。
